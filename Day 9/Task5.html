Box-Sizing: Content-Box vs Border-Box

box-sizing: content-box (default) The width and height apply only to the content area. Padding and borders are added on top, making the total box size larger. So, total width = content + padding + border
box-sizing: border-box The width and height include padding and border. The total size stays fixed, and the content area shrinks to make room for padding and border. So, total width = declared width.

Both boxes have the same width (600px), padding (30px), and border (10px), but notice content-box is larger because the padding and border are added on top of the set width and height.Border-box keeps the total size fixed at 600px by including the padding and border inside it.
