Three Main Terms Of HTML.

Tag:
  1. A tag tells the browser what kind of content it is.
  2. Tags are always inside angle brackets < >.

There are two types of HTML tags:
  • paired Tags:Have an opening tag and cloising tag
  • Self Tags:Don't need a closing tag.

Element:
  1. An element is everything from the start tag to the end tag ,incliding the content inside.

There are three types of Elements in HTML:
  • Block level Element:Always start on a new line and take the full width available.
  • Inline Element:Do not start on a new line and Only take as much width as needed.
  • Empty Element:Have no content and no closing tag.

Attribute:
  1. An attribute gives extra information about an HTML element.
  2. It is always written inside the opening tag.

Examples of Attribute:
  • herf attribute in linlks:tells the browser where the link goes.
  • src attribute in images:specifies the image file.
