Display Property.

1️⃣ Display;Block

Always start on a new line.
Occupy the entire width of the container (even if content is smaller).
You can set width, height, margin, and padding.

2️⃣ Display;Inline

Don't start on a new line.
You can't set width or height.
You can use padding and margin, but they affect layout differently (especially vertical ones).

3️⃣ Display;Inlne-block

Does not start on a new line (like inline).
Can have width and height (like block).
Very useful for creating button-like elements or layouts in a line.

4️⃣ Display;None

It's not visible and takes up no space.
Often used to hide elements temporarily with JavaScript or CSS.
