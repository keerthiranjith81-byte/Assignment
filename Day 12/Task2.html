Floating an image beside text

When an image is floated beside text using the CSS property float, it allows the text to wrap naturally around the image, creating a clean and magazine-style layout. However, floating an element takes it out of the normal document flow, which can cause the parent container's height to collapse. To fix this, a technique called clearfix is used. The clearfix adds a hidden element after the floated items to clear the float and make the parent container expand properly. This method helps maintain the overall layout structure while keeping the image and text neatly aligned.

When children inside a container are floated, the parent's height may collapse because floats are taken out of normal flow. The clearfix (using ::after) creates an invisible block after the children and clears the floats so the parent expands to contain them.

float: left (or right) takes the image out of normal flow and lets inline content wrap around it. When children are floated, the parent's height can collapse (because floats are taken out of the normal flow). The clearfix forces the parent to contain those floats by adding an invisible block after the children that clears the float. The clearfix used here is the well-known ::after technique.
